Web is cool

Web is cool, because it provides useful informations and connecting the world.

A web page is commonly consists of three main parts:

  • HTML - HyperText Markup Language
  • CSS - Cascading Style Sheets
  • JAVASCRIPT - Born for web, lived everywhere

With those technologies I build this web page. Thanks to MDN it's not that hard.

By coding this web page, I learned several things about web, here are some:

  • web pages using hyperlink to connect each other
  • resources are linkable using an URL
  • different content using different tag, for semantic
  • html is for content, css is for styling, js is for behavior
  • and many more...

Web techs used in this page are very basic, no framework and library. I know there are many things to learn to build a serious website. We dont always need to be serious, have fun is important.
